# VS Code Dark+ 主题统一 — 设计方案

## 1. 概述

- **目标**: 将工作流引擎 UI 从「复古工业风 × 赛博朋克暖光」完全切换到 **VS Code Dark+** 风格
- **参照**: VS Code Dark+（默认深色主题）
- **实现方式**: 替换 `theme.py` 中 `'dark'` 主题的颜色常量，调整所有 QSS 样式

## 2. 颜色系统

### 2.1 框架色（从暖色 → VS Code 冷色）

| 用途 | 替换前 | 替换后 |
|------|--------|--------|
| 主强调 (`primary`) | `#E8A87C`（琥珀橙） | `#007ACC`（VS Code 蓝） |
| 次强调 (`secondary`) | `#C38D9E`（珊瑚） | `#094771`（深蓝） |
| 高亮 (`accent`) | `#F5C07A`（暖黄） | `#0098FF`（亮蓝） |
| 主文字 (`text`) | `#E0D8D0` | `#D4D4D4` |
| 背景/面板/边框色 | 不变 | 不变（已匹配 VS Code Dark+） |

### 2.2 节点分类色（降饱和 40%）

| 分类 | 替换前 | 替换后 |
|------|--------|--------|
| 输入 | `#4A90D9` / `#2E6EB0` | `#6A9EC9` / `#4A7EA9` |
| 输出 | `#5CB85C` / `#3D8B3D` | `#7A9E6B` / `#5A7E4B` |
| 流程控制 | `#E8A87C` / `#C38D9E` | `#C8A080` / `#A88060` |
| 数据处理 | `#D4A574` / `#B8860B` | `#B0A07A` / `#90805A` |
| AI | `#9B59B6` / `#7D3C98` | `#9A7AAA` / `#7A5A8A` |

## 3. 组件样式变更

### 3.1 状态栏

- **背景**: `#007ACC`（取代 `#252526`）
- **文字**: `#FFFFFF`

### 3.2 面板标题

- **左边框**: 3px `#007ACC`（取代 `#E8A87C`）

### 3.3 菜单栏 / 菜单

- **悬停**: 背景 `#094771`（取代 `#E8A87C30`）
- **按下**: 背景 `#007ACC`，文字 `#FFFFFF`
- **右键菜单选中**: `#094771` 背景

### 3.4 工具栏

- **按钮悬停**: 背景 `#094771`，图标变蓝
- **按钮激活**: 背景 `#007ACC`

### 3.5 按钮

- **悬停**: 背景 `#007ACC`，文字 `#FFFFFF`
- **Primary 按钮**: 背景 `#007ACC`

### 3.6 选中/聚焦

- **输入框聚焦边框**: `#007ACC`
- **树形控件选中**: 背景 `#094771`
- **标签页选中**: 边框 `#007ACC`
- **节点选中发光**: `#007ACC` 替代 `#E8A87C`

### 3.7 滚动条悬停

- `#007ACC` 替代 `#E8A87C`

## 4. 实施计划

### 涉及文件

| 文件 | 操作 | 修改内容 |
|------|------|----------|
| `ui/styles/theme.py` | 修改 | `THEMES['dark']['colors']` 全部替换为新值 |
| `ui/styles/base.qss` | 修改 | 悬停/选中色变量无需改（已用 @变量） |
| `ui/styles/components.qss` | 修改 | 节点面板/属性面板/日志面板的 `@primary` 无需改 |
| `ui/styles/main_window.qss` | 修改 | 工具栏按钮激活色无需改 |
| `ui/canvas.py` | 修改 | `NODE_COLORS` 替换为降饱和版本，节点发光色替换 |
| `ui/node_panel.py` | 修改 | `NODE_CATEGORIES` 颜色替换为降饱和版本 |
| `ui/main_window.py` | 修改 | 状态栏背景色设置 |

### 风险

- **低风险**: QSS 全部用 `@variable` 变量，`base.qss`、`components.qss`、`main_window.qss` 无需修改
- **中风险**: `canvas.py` 和 `node_panel.py` 中硬编码的颜色需手动替换（共约 10 处）
- **非阻塞**: `main_window.py` 状态栏背景色需添加 `setStyleSheet`
