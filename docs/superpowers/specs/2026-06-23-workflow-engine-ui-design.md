# 工作流引擎 UI 美化设计方案

## 1. 概述

- **项目名称**: 工作流引擎 UI 美化
- **目标**: 将现有的朴素 PySide6 Fusion 风格工作流编辑器，升级为具有**复古工业风 × 赛博朋克暖光**特色的专业可视化界面
- **用户**: 工作流引擎开发者及最终用户
- **设计风格**: 专业深色主题 + 暖色系（深灰背景 + 琥珀/橙色点缀）

## 3. 组件设计方案

### 3.1 主窗口 (MainWindow)

- **背景**: `#1E1E1E`
- **菜单栏**: `#252526` 背景 + `#E0D8D0` 文字，悬停时 `#3C3C3C` 背景
- **工具栏**: `#252526` 背景 + 底部 1px `#3C3C3C` 分割线
- **状态栏**: `#252526` 背景 + 左侧 `#E8A87C` 指示条
- **布局分隔**: 面板之间 1px `#3C3C3C` 分割线

### 3.2 节点面板 (NodePanel) — 扁平网格卡片

- **面板背景**: `#2D2D2D`
- **标题**: 14px 加粗 `#E0D8D0`，左侧 3px `#E8A87C` 装饰条
- **卡片网格**: 2列布局，卡片间距 8px，内边距 8px
- **卡片样式**:
  - 尺寸: 100px × 70px，圆角 6px
  - 背景: 类别对应主色，`border: 1px solid` 类别副色
  - 悬停: `brightness(1.15)` + 底部 2px 琥珀高亮条
  - 图标: 居中，24px，类别对应色
  - 文字: 底部居中，10px，`#E0D8D0`
- **拖拽**: 卡片悬停时添加 `+` 光标，拖出时半透明副本

### 3.3 画布 (WorkflowCanvas) — 网格点阵背景

- **背景**: `#1E1E1E`
- **网格点阵**: 20px 间隔，`#2A2A2A` 圆点，直径 1.5px
- **缩放范围**: 25% ~ 400%，滚轮缩放 + 工具栏缩放按钮

### 3.4 节点 (NodeGraphicsItem) — 圆角卡片 + 发光呼吸

- **尺寸**: 140px × 80px，圆角 8px
- **背景**: 类别对应渐变色（从上到下 10% 亮度渐变）
- **边框**: 1px 类别副色，悬停时 2px `#E8A87C`
- **选中**: 外发光效果（`QGraphicsDropShadowEffect`，`blurRadius=15`，`color=#E8A87C40`）
- **执行中状态**:
  - 外发光脉冲：`blurRadius` 在 15~30 之间以 1Hz 频率正弦波动
  - 边框颜色在 类别副色 → `#F5C07A` 之间脉冲
- **已完成状态**: 边框变 `#5CB85C`，发光消散
- **失败状态**: 边框变 `#E74C3C`，持续红色发光
- **端口**:
  - 输入: 左侧，圆形，半径 6px，`#5CB85C`
  - 输出: 右侧，圆形，半径 6px，`#E74C3C`
  - 悬停: 放大 1.3x，亮度提升

### 3.5 连线 (EdgeGraphicsItem) — 贝塞尔曲线

- **普通**: 2px 宽，`#5C6370`，贝塞尔曲线
- **悬停**: 2px 宽，`#E8A87C`，轻微发光
- **执行中**: 2px 宽，`#F5C07A`，沿连线方向的流动光效（QTimer 动画，10Hz）
- **已完成**: 2px 宽，`#5CB85C`
- **失败**: 2px 宽，`#E74C3C`
- **贝塞尔控制点**: 自动计算，曲线凸向远离连线中垂线的方向

### 3.6 属性面板 (PropertyPanel)

- **面板背景**: `#2D2D2D`
- **标题**: 14px 加粗 `#E0D8D0`，左侧 3px `#E8A87C` 装饰条
- **分组框 (QGroupBox)**: 标题 `#E8A87C`，边框 `#3C3C3C`
- **输入框 (QLineEdit)**:
  - 背景: `#1E1E1E`，边框 `#3C3C3C`
  - 聚焦: 边框 `#E8A87C`
  - 圆角: 4px
- **按钮**:
  - 主按钮: 背景 `#E8A87C`，文字 `#1A1A1A`
  - 次按钮: 背景 `#3C3C3C`，文字 `#E0D8D0`
  - 悬停: 亮度 +10%
  - 圆角: 4px

### 3.7 日志面板 (LogPanel)

- **面板背景**: `#1E1E1E`
- **标题**: 14px 加粗 `#E0D8D0`，左侧 3px `#E8A87C` 装饰条
- **日志文字区**: 等宽字体 `#C0C0C0`，11px
- **日志级别颜色**:
  - INFO: `#4EC9B0` (青绿)
  - DEBUG: `#9CDCFE` (浅蓝)
  - WARNING: `#DCDCAA` (暖黄)
  - ERROR: `#F14C4C` (红)
  - SUCCESS: `#6A9955` (绿)
  - 时间戳: `#808080` (灰)
- **清除按钮**: 右上角，图标按钮风格

### 3.8 工具栏 (Toolbar) — 专业图标 + 文字

- **图标**: 使用 Qt 内置图标（标准符号），统一线性风格
  - 新建: `SP_FileIcon` 或自绘
  - 运行: `SP_MediaPlay`（琥珀色）
  - 暂停: `SP_MediaPause`
  - 停止: `SP_MediaStop`
  - 单步: `SP_MediaSkipForward`
  - 重置: `SP_BrowserReload`
  - 缩放: `SP_ZoomIn` / `SP_ZoomOut`
- **图标尺寸**: 20px × 20px
- **文字**: 图标下方，9px，`#8A8A8A`
- **按钮悬停**: 背景 `#3C3C3C`，图标颜色变 `#E8A87C`
- **分隔符**: 1px 竖线 `#3C3C3C`

### 3.9 菜单栏 (MenuBar)

- **背景**: `#252526`
- **文字**: `#E0D8D0`
- **悬停项**: 背景 `#3C3C3C`
- **子菜单**: `#2D2D2D` 背景 + `#3C3C3C` 边框
- **选中项**: 背景 `#E8A87C20`（半透明琥珀）
- **快捷键文字**: `#8A8A8A`

### 3.10 右键上下文菜单

- **背景**: `#2D2D2D`
- **边框**: 1px `#3C3C3C`，圆角 4px
- **阴影**: `QGraphicsDropShadowEffect`，`blurRadius=10`
- **菜单项**:
  - 默认: `#E0D8D0`
  - 悬停: 背景 `#E8A87C30`
  - 分隔线: 1px `#3C3C3C`

### 3.11 对话框 (QDialog / QMessageBox)

- **背景**: `#2D2D2D`
- **标题栏**: `#252526`
- **按钮**: 同属性面板按钮样式
- **圆角**: 8px

### 3.12 滚动条 (QScrollBar)

- **轨道**: `#1E1E1E`
- **滑块**: `#3C3C3C`，悬停 `#5C6370`
- **箭头按钮**: 隐藏或 `#3C3C3C`

## 4. 动画效果规范

### 4.1 节点呼吸脉冲

- **触发**: 节点执行中（状态=running）
- **频率**: 1Hz（1秒一个周期）
- **参数变化**:
  - `QGraphicsDropShadowEffect.blurRadius`: 15 → 30 → 15（正弦）
  - 边框宽度: 1px → 2px → 1px
  - 边框颜色: 类别副色 → `#F5C07A` → 类别副色

### 4.2 连线流动光效

- **触发**: 连线处于执行中状态
- **实现**: QTimer 10Hz，沿贝塞尔曲线移动一个高亮点
- **颜色**: `#F5C07A`，透明度 60%

### 4.3 面板悬停过渡

- **时长**: 100ms
- **效果**: `background-color` 渐变

## 5. 文件结构

```
workflow_engine/ui/
├── __init__.py
├── main_window.py      # 主窗口（已存在，保持不变）
├── canvas.py           # 画布（重构贝塞尔曲线、节点样式、背景）
├── node_panel.py       # 节点面板（重构为网格卡片）
├── property_panel.py   # 属性面板（美化样式）
├── log_panel.py        # 日志面板（美化样式）
├── styles/             # 新增
│   ├── __init__.py
│   ├── theme.py        # 主题常量定义
│   ├── base.qss        # 基础全局样式
│   ├── main_window.qss # 主窗口样式
│   ├── canvas.qss      # 画布样式
│   ├── node_panel.qss  # 节点面板样式
│   ├── property_panel.qss # 属性面板样式
│   ├── log_panel.qss   # 日志面板样式
│   └── components.qss  # 通用组件样式（按钮/输入框/滚动条等）
└── icons/              # 新增（如需自定义图标）
    └── (SVG/PNG 图标文件)
```

## 6. 实现优先级

1. **P0 - 基础样式文件**: `theme.py` + `base.qss` + `components.qss`
2. **P0 - 节点面板网格化**: `node_panel.py` + `node_panel.qss`
3. **P0 - 画布美化**: 网格点阵背景 + 贝塞尔曲线连线 + 节点样式
4. **P1 - 节点呼吸动画**: 执行状态下的脉冲发光效果
5. **P1 - 工具栏图标**: Qt 标准图标 + 文字标签
6. **P1 - 属性/日志面板美化**: 统一配色
7. **P2 - 菜单/右键菜单美化**: 统一配色
8. **P2 - 对话框美化**: 圆角 + 配色
